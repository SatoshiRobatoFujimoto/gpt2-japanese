# gpt2-japanese
Japanese GPT2 Generation Model

# GPT2日本語モデル

## GPT2に日本語Wikipedia全文コーパスを学習させたものです

## 日本語文章の生成モデルです

- パラメーター数117Mの一番小さなモデルのみです
- より大きなパラメーター数のモデルは、気が向いたら作るかもしれません
- 作らないかもしれません
- 日本語Wikipediaが元なので、Wiki記事っぽい文章が生成されます
- 英語版のGPT2のような、フェイクニュースっぽい記事は作れません
- ニュース記事を元にしたコーパスは学習させていないからです
- パラメーター数が小さなモデルだからか、けっこう雑な結果です
- 文章のベクトル化とかには使えるかもしれません

### 使い方

```
$ cat ja-117M/model-1339900.data-00000-of-00001.00 ja-117M/model-1339900.data-00000-of-00001.01 ja-117M/model-1339900.data-00000-of-00001.02 ja-117M/model-1339900.data-00000-of-00001.03 ja-117M/model-1339900.data-00000-of-00001.04 >  ja-117M/model-1339900.data-00000-of-00001
$ python3 gpt2-generate.py --num_generate 1
```
